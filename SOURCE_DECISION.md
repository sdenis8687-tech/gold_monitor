# Source Decision

## Финальное решение
В проекте используется только один production source:
- `MOEX ISS`

## Production instruments
Используются только:
- `GLDRUB_TOM`
- `USD000UTSTOM`

## Почему принято именно это решение
- источник официальный;
- доступ к delayed данным есть без платного API в рамках текущего технического решения;
- золото и USD/RUB доступны в одном источнике;
- нет необходимости в HTML scraping;
- нет зависимости от нестабильной верстки сторонних сайтов.

## Scope decision
Этот проект описан как:
- внутренний инструмент мониторинга для заказчика;
- не публичный data-product;
- не сервис перепродажи данных;
- не multi-tenant продукт.

## Запрещенные варианты
Запрещено:
- использовать Kitco как источник;
- парсить HTML Kitco;
- использовать 1000bankov как production source;
- использовать платные API;
- менять source strategy.

## Expected result
В результате должен получиться:
- юридически и технически более аккуратный внутренний dashboard;
- полностью отдельный кодовый продукт вместо n8n workflow;
- стабильная загрузка данных из одного официального источника.
